Icomoon icon set

Icomoon is a custom icon set generated in IcoMoon app, includes 1145 icon glyphs based on 16px grid. Starting from version 4.0 of the template, Icomoon is now an optional icon set along with Material and Font Awesome sets that are also included in the template. All icons are retina-ready by default, because majority of components are using font family for UI elements instead of images, so regular text styles are also applied to icons.
